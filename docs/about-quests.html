Quests

Das Spiel wird mehr oder weniger ein Open World Spiel werden. Erklärtes Ziel wird es aber sein neue Wege des Storytellings zu erproben um den Spielern eine glaubwürdige Welt mit sich entwickelnder Geschichte präsentieren zu können. Dabei muss man sich von dem Gedanken verabschieden, dass jeder Spiele jede Teilgeschichte/Quest durchleben kann. Dies widerspricht dem persistenten und inneren logischem Aufbau der Spielwelt. Quests müssen sich vielmehr dynamisch aus dem geschehen innerhalb der Spielwelt ergeben. Einige Beispiele:

Hilf dem Bauern!

Eine reiche Ernte steht bevor, aber der Bauer Wilhelm kann aufgrund seiner knappen Zeit nicht die ganze Ernte einfahren bevor sie auf den Feldern verdirbt. Er fragt vorbeiziehende Spieler ob sie ihm nicht dabei helfen könnten. Dies wird mit einem EXP Bonus, Gold und ggf. einem Item belohnt.

Verteidigt die Stadt!

Aufgrund hoher Rift-Aktivität machen viele hochstufige Bestias die Umgebung einer großen Stadt unsicher. Der Spieler wird aufgefordert die Bestias zu bekämpfen, er steigt in der Gunst der Stadt auf und erhält Zugang zu besseren Gegenständen, Erfahrung und Gold.

Hilf mir!

Der NPC hat sich in einem gefährlichen Gebiet verlaufen und versteckt sich nun. Findet ein Spieler den NPC so kann er ihm ggf. helfen seinen Weg zurück zu finden und wird darauf mit EXP und Gold belohnt.

Quests können auf zwei verschiedene Arten erstellt werden:

  1. Vom Spiel automatisch generierte Quests

  2. Vom Spieler ausgeschriebene Quests, diese müssen an NPC vergeben werden, samt Belohnung, die diese Quests "ausschreiben". (Schilder können keine Belohnungen aushändigen, wenn ein Quest erfolgreich abgeschlossen wurde).

Von Spielern erstellte Quests

Spieler können diverse Aufgaben, wie zum Beispiel das Sammeln von Rohstoffen oder das besiegen bestimmter Gegner, oder den Transport bestimmter Waren als Aufgabe definieren und eine Belohnung ausloben. Auch das Abstellen von Bestias als Wächter wäre eine denkbare Aufgabe. Diese Belohnung muss dann an die entsprechenden NPCs abgegeben werden, die diese dann treuhänderisch verwahren (oder einfach untereinander unter den Spielern verhandelt werden).

Das ausloben von Quests muss sich für beide Parteien lohnen. Der Spieler bekommt sowohl für das Ausschreiben sowie für das von einem anderen Spieler erfolgreiche Abschließen der Quest eine gute Menge Erfahrung zugeschrieben. Diese kann er unter Umständen (Items benötigt) auf Bestias in seinem Besitz umverteilen. Beim Annehmen einer Quest wird ein Vertrag zwischen Spieler und NPC geschlossen. Sollten alle Vertragsbedingungen erfüllt werden, so wird die Belohnung ausgehändigt. Verträge können zeitlich begrenzt sein.
