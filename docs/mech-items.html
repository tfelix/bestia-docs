Items

Each item has a level assigned. This level is important to determine how hard or easy it is to forge this item, upgrade it and so on.

Weapon refinement

Weapons can be refined which accounts to extra damage dealt if the weapon is used. Each upgrade step can result in a destruction of the equipment.

Equipment refinement

An equipment can be refined up to +10. Each higher refinement process can destroy the weapon/equipment. The chance of success is depending on skill and refine items used. The refinement bonus depending on level has not yet been decided.

A refined armor has increased resistance for hard def. Each upgrade point grants ⅓ more defense.

A refined weapon has increased damage.

Item recycling

In order to produce raw materials existing items can be taken apart. This will generate the raw materials from which they were produced originally.

Item crafting

Bestia uses an innovative new item crafting. Basically the player is allowed to craft all items. Some items however might be so hard to craft so it is not possible to craft it without appropriate skill. Some items may require special ingredients which can not be substituted. The higher the crafted item level and the harder it get to successful craft it. If the crafting of the item fails all the materials used to craft it are lost. In order to craft an item two things are important: the sum of raw materials used to craft and how the materials are laid out.

Item craft success chance is determined if the raw materials

totalAmoun = Summe i N Min(1, i/amount) / N

Wenn totalAmount (tA) < 0.5 → SuccessChance = 0

totalAmount (tA) > 0.5 → SuccessChance = e^-(5*(1 - tA)) * (craftSkill * 10 / itemLevel) * craftMod

After an item was successfully crafted the recipe is saved for the user inside his recipe list. From this list the recipe can be inscribed upon a paper to give it to other players who can consume and learn it. The learning will take some time (depending of the level of the item).

Data structure for a recipe: {rows: n, columns: m, items: [id-amount,id-amount,...], result: itemId}

Related: See crafting skill.

Alchemy brewing

The brewing is a time consuming process. The process should be somehow deterministic so the results are logically to be obtained.

Once a receipt was discovered it is saved for the user so it can be reused very quickly. If a player repeatedly makes the same receipt the quality of the produced potions increases over time. The player can choose to write down the receipt and give it to other players who might increase their skill by reading it.

Craft duration

Craft duration is determined by the item level which the player wants to craft. Skill points in the suitable skill tree (weapon, equipment, construction or alchemy, artifacts) reduces the build time.

Shortest duration is: 3 min (Lv. 1) up to 2 days (Lv.100).

Item status values

Items are potentially able to receive attacks while they are on the ground. Thus to participate in the battle system the need status values. Items status value distribution is based upon their type (weapon, potion, book etc) and the total amount of points to distribute is given by the item level. Later there might be items which get preset custom values. But for now on they are based upon a lookup table.

Items und Entities Interaktion

Es sollte möglich sein Items mit Entities zu verknüpfen. Eine Verschachtelung sollte also möglich sein. Mit dieser Technik wird es erlaubt aktive Items zu kombinieren. Zum Beispiel:

  • Energiekristalle an ein magisches Artefakt anbringen bzw. entnehmen.

  • Kisten die mit Inhalt ausgestattet werden können.

Zu diesem Zwecke müssen die Items limitiert werden können die in eine Entity gelagert werden können. Das Filtern muss sehr flexibel sein, bestimmte Items ausschließen können, das Gewicht limitieren können etc. Jeder Gegenstand besitzt auch eigene Widerstandswerte. Sobald eine Bestia getötet wird wird überprüft ob und wenn ja wieviel Schaden einzelne, getragene Gegenstände nehmen. Auch Gegenstände die auf dem Boden liegen können durch Attacken angegriffen und ggf. zerstört werden. Items aus Papier sind zum Beispiel anfälliger als Rüstungsteile aus Metall.

Zerstörung von Items

Sogut wie jedes Item in Bestia kann beschädigt werden. Es wird entsprechende Berufe geben die in der Lage sind Gegenstände wieder zu reparieren. Gegenstände funktionieren in einem Weiten Rahmen ihres Zustandes normal. Ab einem gewissen Grad verlieren sie langsam ihre Eigenschaften (Rüstungen ihren Rüstwert, Waffen treffen seltener, Magische Stäbe erleiden einen Schadensmalus usw.). Bis die Gegenstände unbenutzbar werden. Wenn man ihnen dann weiter Schaden zufügt können sie komplett und unwiederbringlich zerstört werden.
